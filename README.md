# AI Agent NestJS + React Boilerplate

This repository is a **monorepo boilerplate** for building an AI-powered agent using:

- **TypeScript**
- **NestJS** (backend, API + agent logic)
- **React** (frontend chat UI)
- **Shared types & utilities** (packages/shared)

> The boilerplate is included.  
> Clone it, install dependencies, and start building your own AI agent.

## Monorepo structure

```txt
apps/
  server/   # NestJS backend (coming next)
  web/      # React frontend (coming next)
packages/
  shared/   # Shared TypeScript types & utils
